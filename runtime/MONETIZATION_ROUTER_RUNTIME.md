# Monetization Router Runtime v1.0

## 判定
- product_reference_urlまたはaffiliate_urlがある: AFFILIATEを優先。
- 口コミ、評判、レビュー、比較、おすすめ、料金、申込み、解約、教材、商品、サービスの購入判断検索: AFFILIATE候補。
- 操作、設定、解決、意味、費用計算、一般解説: ADSENSE候補。

## 出力
monetization.typeをADSENSE / AFFILIATE / HYBRID / NONEから選ぶ。

## AFFILIATE処理
商品特定、URL調査、重要条件、口コミ区分、PR表記、CTA挿入を計画する。

## ADSENSE処理
問題解決と内部回遊を優先し、商品CTAを無理に追加しない。
