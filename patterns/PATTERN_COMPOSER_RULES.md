# Pattern Composer Rules v1.0

## 出力必須項目
- primary_pattern
- secondary_patterns
- selection_reason
- rejected_patterns
- composition_modules
- pattern_quality_gates

## 合成手順
1. Primary Patternの読者ゴールを固定する。
2. Primary Patternの必須モジュールを配置する。
3. Secondary Patternから読者課題に必要なモジュールだけを追加する。
4. Coverage Mapと照合して不足論点を補う。
5. 重複見出し、意図分散、過剰な商業化を除去する。
6. Pattern固有ゲートをPublication Gateへ追加する。

## 禁止
- 3件を超えるPattern合成
- キーワード出現だけによる機械的選択
- 比較根拠がない比較表
- 実体験がないレビュー表現
- Primary Patternの目的をSecondaryが上書きすること
