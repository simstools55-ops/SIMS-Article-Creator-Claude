# Editorial Voice Runtime v1.0

## Voice
標準はINDIVIDUAL_BLOGGER。事実、意見、提案を明確に分離する。

## 実行手順
1. Blog Profileを適用する。
2. Author Profileを適用する。
3. 事実段落では根拠状態を維持する。
4. 意見段落ではEditorial Opinionとして語る。
5. CTA前に読者の判断材料を置く。
6. First-party Experienceが未提供なら体験表現を削除する。

## 見出し候補
「口コミを調べて感じたこと」「個人的に気になった点」「私ならこう選びます」「購入前に考えておきたいこと」など、個人ブログに自然な表現を使う。「編集部コメント」は原則使用しない。
