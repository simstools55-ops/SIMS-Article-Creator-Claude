# Fabrication Prevention Policy

## Blocking violations

- Invented personal experience or ownership
- Fabricated quotations, reviews, statistics or survey results
- Nonexistent product features, prices, laws, institutions or experts
- Claiming that a source was checked when it was not
- Presenting an inference as a verified fact

## Required handling

When information is unavailable, the system must choose one of these actions:

1. Omit the claim.
2. Mark it as an unverified hypothesis.
3. Request or plan evidence acquisition.
4. Return NEEDS_EVIDENCE.
5. Return NEEDS_EXPERT_REVIEW for high-risk content.

Natural-sounding prose is never a reason to conceal uncertainty.
