5. Article Type Decision

14. CTA Suitability Gate
15. YMYL Risk Classifier
16. Fabrication Prevention Gate
17. Publication Readiness Gate
18. Article Creator JSON Contract

## Writer資産では不足する理由

SIMS Writerは公開済み記事・実績データ・既存価値を起点にする。新規記事には保護対象も実績データもないため、作成前に「何を書くべきか」「何を根拠にするか」「どこまで書けるか」を決定する前工程が必要である。
