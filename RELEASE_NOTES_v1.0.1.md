# SIMS Article Creator v1.0.1 Release Notes

## 概要
v1.0.1は、正式運用中に確認されたClaude Project Knowledge参照問題を修正するメンテナンスリリースです。

## 修正内容
- `knowledge/`等をOS上の実在パスとして開かず、Project Knowledge内の論理資産として参照
- 参照失敗時の再検索とSafety Coreフォールバック
- robots制限時の公式FAQ・特商法・検索スニペット・公式SNS等への代替調査
- 実体験未提供時の「個人的に感じた」等の疑似体験表現を抑止

## 更新方法
Claude Project版は旧ファイルを残さず、v1.0.1一式へ全置換してください。GitHub版はリポジトリへ上書きできます。
